= Verify the Cluster

Your SNO cluster should now be fully operational. Let's verify everything is working.

== Cluster Version

[source,bash,role=execute]
----
oc get clusterversion
----

Expected: `VERSION 4.22.0`, `AVAILABLE True`, `PROGRESSING False`.

== Node Status

[source,bash,role=execute]
----
oc get nodes
----

Expected: One node (`cp-0`) with `STATUS Ready` and roles `control-plane,master,worker`.

== Cluster Operators

[source,bash,role=execute]
----
oc get co
----

All operators should show `AVAILABLE True`, `PROGRESSING False`, `DEGRADED False`.

== Quick Health Check

[source,bash,role=execute]
----
# Count operators by status
echo "Total operators: $(oc get co --no-headers | wc -l)"
echo "Available:       $(oc get co --no-headers | awk '$3=="True" {n++} END {print n}')"
echo "Degraded:        $(oc get co --no-headers | awk '$5=="True" {n++} END {print n}')"
----

== Eject the Config ISO

Now that the cluster is up, clean up by ejecting the config ISO:

[source,bash,role=execute]
----
BMC_IP=$(grep 'bmc_ip' ~/ibi-lab/bmc-info.txt | awk '{print $2}')
BMC_USER=$(grep 'bmc_user' ~/ibi-lab/bmc-info.txt | awk '{print $2}')
BMC_PASS=$(grep 'bmc_pass' ~/ibi-lab/bmc-info.txt | awk '{print $2}')
SYSTEM_ID=$(curl -s -u "${BMC_USER}:${BMC_PASS}" \
  http://${BMC_IP}:8000/redfish/v1/Systems \
  | python3 -c "import json,sys; print(json.load(sys.stdin)['Members'][0]['@odata.id'].split('/')[-1])")

curl -s -u "${BMC_USER}:${BMC_PASS}" \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{}' \
  http://${BMC_IP}:8000/redfish/v1/Systems/${SYSTEM_ID}/VirtualMedia/Cd/Actions/VirtualMedia.EjectMedia

echo "Config ISO ejected"
----

=== Key Takeaways

* **IBI is fast** — ~20 minutes from blank server to running cluster
* **Redfish is the standard** — the same API works on iDRAC, iLO, Supermicro, and virtual BMC
* **Seed images are reusable** — one image deploys to hundreds of identical edge sites
* **recert makes each cluster unique** — certificate regeneration ensures no two clusters share keys
